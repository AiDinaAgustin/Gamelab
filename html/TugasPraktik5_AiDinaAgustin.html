Hello, Aku Dina

Aku adalah mahasiswa semester 6 prodi Teknik Informatika. Dan aku sekarang sedang menjalankan kegiatan Studi Independen di Gamelab Indonesia diposisi Backend Web Developer. Aku mulai menyukai Programming ketika SMK, waktu itu aku masuk Jurusan Rekayasa Perangkat Lunak. Setelah menyelesaikan kelas ini, saya ingin menjadi seorang web developer yang handal dan dapat menciptakan aplikasi web yang bermanfaat bagi orang lain. Saya juga ingin terus belajar dan berkembang di bidang ini.

I Like, GameLab

Awal mula saya tertarik dengan dunia web development karena ketertarikan saya pada teknologi dan keinginan untuk menciptakan sesuatu yang bermanfaat bagi orang lain. Saya memilih Gamelab karena reputasi mereka yang baik dalam industri game dan teknologi, serta kurikulum yang dirancang untuk industri. Kelas web developer di Gamelab dirancang untuk memberikan pemahaman menyeluruh tentang front-end dan back-end development. Saya belajar tentang HTML, CSS, JavaScript, PHP, dan berbagai framework populer.
